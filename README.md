# Travel Planner

## Herramientas

- Node.js v22.20.0
- npm v10.9.3
- Git

## Primeros Pasos

1. Clonar el repositorio:
   ```bash
   git clone git@github.com:imjimenez/travel-planner.git
   ```
2. Instalar las dependencias:
   ```bash
   cd travel-planner
   npm install
   ```
3. Ejecutar el proyecto:
   ```bash
   npm start
   ```
4. Abrir el navegador en http://localhost:4200
